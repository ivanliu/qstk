get_data_hardread(self, ts_list, symbol_list, data_item, verbose=False, bIncDelist=False)

source code 

Read data into a DataFrame no matter what.

Parameters:
  • ts_list - List of timestamps for which the data values are needed. Timestamps must be sorted.
  • symbol_list - The list of symbols for which the data values are needed
  • data_item - The data_item needed. Like open, close, volume etc. May be a list, in which case a list of DataFrame is returned.
  • bIncDelist - If true, delisted securities will be included.

Note: If a symbol is not found then a message is printed. All the values in the column for that stock will be NaN. Execution then continues as usual. No errors are raised at the moment.
