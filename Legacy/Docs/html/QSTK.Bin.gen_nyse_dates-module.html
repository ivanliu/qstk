Module gen_nyse_dates

source code

Created on March 22, 2011

Author: Tucker Balch

Contact: tucker@cc.gatech.edu

This script generates a list of days on which we expect the NYSE to be open. Information is drawn from two websites:

http://www.timeanddate.com/holidays/ http://www.nyse.com/about/newsevents/1176373643795.html
